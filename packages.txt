To install the packages run " npm install --save <name of package> " command (except nodemon) in your terminal.

Use " npm install --save-dev nodemon" command to install nodemon package.

Here are some Packages you have to install in your Device:
    
    
    "body-parser"
    "ejs"
    "express"
    "express-handlebars"
    "nodemon"
    "path"
    "mysql2"
    "sequelize"
